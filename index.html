    Guide

    Help

    The Angle
    左の強化で周回速度、角、頂点獲得量を伸ばします。
    世代
    世代スコアが 1,000,000 に届いたら、下部の世代交代で倍率を得ます。
    核増幅
    1.00e20 スコアから実行でき、世代以下をリセットして指数と増加倍率を得ます。
    Infinity
    1.80e308 到達で初回は自動発動し、以降は任意発動で IP を得ます。
    Infinity Challenge
    Infinity 後に挑戦できます。縛り状態で Infinity に到達すると報酬を得ます。
    Break Infinite Cap
    1.00e350 到達で、Infinity 以降の強いソフトキャップを破壊します。
    Infinite Angle
    IP を Infinite Score に変換し、頂点通過ごとの増加を伸ばします。
