# Angle Incremental

**Angle Incremental** は、画面中央の正多角形 **The Angle** を強化してスコアを伸ばしていく、ブラウザ向けのインクリメンタルゲームです。

Point が The Angle の外周を周回し、最上部の頂点である「核」に到達するとスコアを獲得します。スコアを使って周回速度・頂点数・頂点通過時の獲得量を強化し、進行を加速させます。

## 主な要素

- **The Angle**: 初期状態は三角形。強化により頂点数を増やせます。
- **Point**: 図形の外周を移動し、頂点を通過するたびにスコア獲得量を高めます。
- **核**: 最上部の頂点。Point が到達するたびに現在の獲得量に応じたスコアを得ます。
- **Generation**: 累計スコア 1,000,000 で解放されるリセット系の成長要素です。通常進行をリセットする代わりに恒久的なブーストを得ます。

## 起動方法

ビルドや依存関係のインストールは不要です。任意の静的ファイルサーバーでリポジトリのルートを配信してください。

```bash
git clone https://github.com/tara-sooo/Angle_Incremental.git
cd Angle_Incremental
python3 -m http.server 8000
```

ブラウザで `http://127.0.0.1:8000/` を開きます。

## 開発時の確認

JavaScript を変更した場合は、少なくとも構文チェックを実行してください。

```bash
node --check game.js
```

UI またはゲームプレイに影響する変更では、ブラウザでの動作確認に加え、デスクトップ・モバイル表示、セーブデータ、およびコンソールエラーを確認してください。詳細は [CONTRIBUTING.md](CONTRIBUTING.md) を参照してください。
